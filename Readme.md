# Mandelbrot-set
Программа строит изображение множества Мандельброта.

![Пример исполнения программы](doc/img.png)

##
Она присваивает цвет пикселям на экране тремя способами: каждому по отдельности, одновременно группам из восьми пикселей в массиве, одновременно группам из восьми пикселей с испвользованием AVX инструкций.
